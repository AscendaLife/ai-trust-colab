## 🚀 立刻開始（一鍵開啟）

點以下按鈕在 Google Colab 中開啟：

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/AscendaLife/ai-trust-colab/blob/main/AI_Trust_Scanner_Colab.ipynb)

## 📋 使用流程

1. 點上方 "Open In Colab" 按鈕
2. 從上往下執行每個區塊（按 ▶ 按鈕）
3. 在 **步驟 2** 區塊填入您的 License Token、AI 端點、API Key
4. 在 **步驟 3** 區塊勾選要跑的探針
5. 在 **步驟 4** 區塊點 ▶ 開始掃描（5–30 分鐘）
6. 在 **步驟 5** 區塊點 ▶ 下載 PDF + JSON 報告

## 🔒 隱私保證

- 您的 API Key 從 Colab 機器**直接送到您填的 AI 端點**
- AI Trust 的伺服器**完全不介入**，看不到您的金鑰、提問、模型回應
